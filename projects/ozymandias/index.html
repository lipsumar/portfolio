Ozymandias

OPTIMUM PARK™ GitHub

Ozymandias is a nodeJS library to organize tasks among a group of people.
This is the core of the OPTIMUM PARK™ artwork.
Out of the box, this allows you to configure terminals, stations, levels and tasks, register subjects and start a session. A module to simulate the actions of agents and subjects is also provided.
