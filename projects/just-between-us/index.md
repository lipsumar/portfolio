A eurorack clock module heavily inspired by Pam's New Workout. Built to learn electronics and C++, it features two analog outputs, USB MIDI clock sync, a rotary encoder, an OLED display, and a menu-driven UI for setting BPM and per-output modifiers, wave shapes, width, and phase.
